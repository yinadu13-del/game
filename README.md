# hoc-tieng-trung-cung-nana

Website học tiếng Trung cùng Nana, có phần Pinyin nền tảng và các bài học HSK.

## Đưa lên GitHub

```bash
git init
git add index.html hoc-tieng-trung-cung-nana.html README.md
git commit -m "add pinyin learning website"
git branch -M main
git remote add origin https://github.com/yinadu13-del/hsk1.git
git push -u origin main
```

## Ghi chú âm thanh

Âm thanh dùng Web Speech API của trình duyệt với `zh-CN`, ưu tiên các giọng như Microsoft Xiaoxiao hoặc Google 普通话. Khi chạy trên GitHub Pages, hãy dùng Chrome hoặc Microsoft Edge để có khả năng hỗ trợ giọng Trung tốt nhất.
